Receitas de culinária

O que são?

No âmbito da culinária, receitas são orientações, passo a passo, normalmente em forma de listas, de forma a ajudar pessoas a preparar pratos de comida de forma simples. Para além disso, os ingredientes necessários para fazer cada receita encontram-se sempre presentes nas mesmas tal como as respetivas quantidades necessários para cada um.

Que tipo de receitas se podem encontrar nas várias secções deste website?

Neste website irão encontrar vários tipos de receitas, tais como receitas tipicamente portuguesas, receitas saudáveis e até receitas típicas de outros países e continentes (todas devidamente organizadas com os ingredientes necessários e passo a passo com a finalidade de garantir uma simples execução das mesmas).

Onde é possível encontrar informações sobre os autores do website?

Existe uma secção dedicada aos autores com toda a informação necessária acerca dos mesmos sob o nome de "Autores", que pode ser encontrada tanto no fim de todas as páginas como na última opção do menu das mesmas.
